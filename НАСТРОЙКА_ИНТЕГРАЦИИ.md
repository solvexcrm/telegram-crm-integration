# 🤖 Telegram → CRM Интеграция для Facebook лидов

Автоматическая передача лидов из Telegram группы в CRM систему.

## 📋 Что делает система

✅ Получает сообщения от AlbatoBot в Telegram
✅ Парсит данные лидов (имя, телефон, локация)
✅ Автоматически создает лиды в CRM с источником "Facebook"
✅ Логирует все операции для отладки

## 🚀 Установка и запуск

### 1. Установите зависимости
```bash
cd telegram-crm-integration
npm install
```

### 2. Протестируйте парсер
```bash
npm run test
```
Это покажет как система распознает ваши лиды.

### 3. Настройте конфигурацию

В файле `server.js` измените:
```javascript
// Замените на URL вашего CRM API
const CRM_API_URL = 'http://localhost:3000/api/crm';

// Замените на ваш tenant_id из CRM
tenant_id: 1
```

### 4. Запустите сервер
```bash
npm start
# или для разработки:
npm run dev
```

Сервер будет доступен по адресу: `http://localhost:3001`

## 🔧 Настройка Telegram webhook

### 1. Получите токен бота
- Найдите @BotFather в Telegram
- Получите токен вашего бота

### 2. Установите webhook
```bash
# Замените YOUR_BOT_TOKEN на реальный токен
curl -X POST "https://api.telegram.org/bot YOUR_BOT_TOKEN/setWebhook" \
  -H "Content-Type: application/json" \
  -d '{"url":"https://ваш-сервер.com/webhook"}'
```

### 3. Для локального тестирования используйте ngrok
```bash
# Установите ngrok
npm install -g ngrok

# Создайте туннель к вашему серверу
ngrok http 3001

# Используйте полученный URL для webhook
# Например: https://abc123.ngrok.io/webhook
```

## 📊 Формат распознаваемых лидов

Система распознает сообщения в формате:
```
Кладовки исп НОВАЯ ФОРМА[Локация][Имя]+34XXXXXXXXX
```

### Примеры:
- `Кладовки исп НОВАЯ ФОРМАBenicalarDavid Lahorca Avila+34676310032`
- `Кладовки исп НОВАЯ ФОРМАBenicalapaaaron jimenez+34661491177`

### Поддерживаемые локации:
- Angel Guimera
- Benicalap
- Benicalar
- Calle Sant Pedro Pascual,9, Angel Guimera

## 🧪 Тестирование

### 1. Проверьте статус сервера
```bash
curl http://localhost:3001/status
```

### 2. Протестируйте парсер
```bash
curl -X POST http://localhost:3001/test-parser \
  -H "Content-Type: application/json" \
  -d '{"message":"Кладовки исп НОВАЯ ФОРМАBenicalarDavid Lahorca+34676310032"}'
```

### 3. Имитируйте webhook от Telegram
```bash
curl -X POST http://localhost:3001/webhook \
  -H "Content-Type: application/json" \
  -d '{
    "message": {
      "chat": {"id": 123456},
      "text": "Кладовки исп НОВАЯ ФОРМАBenicalarDavid Lahorca+34676310032"
    }
  }'
```

## 📈 Мониторинг

- ✅ Распознанные лиды
- ❌ Ошибки парсинга
- 📤 Отправку в CRM
- 📨 Входящие сообщения

### Проверка в CRM:
1. Откройте CRM
2. Перейдите в раздел "Новые лиды"
3. Найдите лиды с источником "Facebook"

## ⚠️ Устранение неполадок

### Лиды не создаются в CRM
1. Проверьте URL CRM API в `server.js`
2. Убедитесь что CRM API работает
3. Проверьте tenant_id
4. Посмотрите логи сервера

### Webhook не работает
1. Проверьте что сервер запущен
2. Убедитесь что webhook URL доступен извне
3. Проверьте токен бота
4. Посмотрите логи Telegram API

### Парсер не распознает лиды
1. Запустите `npm run test` для проверки
2. Сравните формат ваших сообщений с примерами
3. При необходимости обновите парсер

## 🔐 Безопасность

- Храните токен бота в переменных окружения
- Используйте HTTPS для webhook
- Ограничьте доступ к серверу
- Регулярно обновляйте зависимости

## 📞 Поддержка

Если что-то не работает:
1. Проверьте логи сервера
2. Запустите тестирование парсера
3. Проверьте настройки webhook
4. Убедитесь что CRM API доступно
